## Projects

### 2. Smart-City

<p align="left">
  <img src="https://skillicons.dev/icons?i=python,jupyter" alt="Python, Jupyter" width="80" />
</p>

**Context:** Pre-exploratory data analysis to **optimize maintenance routes** for trees in the city of Paris.

**Deliverables:**
- Exploratory data analysis (EDA) on tree dataset
- Insights to support route planning for maintenance teams
- Jupyter notebook and presentation

---

### 3. Open-Food-Facts

<p align="left">
  <img src="https://skillicons.dev/icons?i=python,jupyter" alt="Python, Jupyter" width="80" />
</p>

**Context:** Exploratory analysis of **Open Food Facts** data with the goal of building a **suggestion or auto-completion system**.

**Deliverables:**
- Exploratory data analysis on product data
- Data cleaning and feature analysis
- Foundation for recommendation / auto-completion
- Jupyter notebook and presentation

### 4. Pret_a_depenser

<p align="left">
  <img src="https://skillicons.dev/icons?i=python,jupyter" alt="Python, Jupyter" width="80" />
</p>

**Context:** Credit scoring project (Prêt à dépenser). **EDA** and **modelling** to predict default risk; dataset from Home Credit.

**Deliverables:**
- EDA notebook (to be run before modelling)
- Modelling notebook: model selection, evaluation, scoring
- Presentation
- Use of provided dataset (download link in project README)

### 5. Olist

<p align="left">

**Deliverables:**
